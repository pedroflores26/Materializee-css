prncipais codigos

aqui mostraremos os codigos mais usados no materialize css

card

Crie cards usando a estrutura div class="card". Coloque um div class="card-content" no card para o conteúdo e um div class="card-action" para os links de ação.

botao

Use a classe waves-effect waves-light btn para criar um botão padrão. Use a tag juntamente com a classe material-icons para adicionar um ícone a um botão. Os botões podem ter vários tamanhos, como btn-large, que é um botão grande.

formularios

Campos de entrada podem ser criados com a classe input-field. Utilize class="validate" para validação de campos. Use textarea com a classe materialize-textarea para áreas de texto maiores. Para botões de submissão, use class="btn waves-effect waves-light" e adicione um ícone com i class="material-icons right".

tabelas

Crie tabelas com a classe striped para linhas alternadas coloridas. Use highlight para destacar a linha ao passar o mouse.

grids

O sistema de grid do Materialize CSS é baseado em 12 colunas. Utilize classes como col s12 para definir a largura das colunas em diferentes tamanhos de tela (small, medium, large).

modais

Para criar modais, utilize div class="modal" e inicialize com JavaScript.

Tooltips

Adicione tooltips com a classe tooltipped e o atributo data-tooltip para o texto do tooltip.

Dropdowns

Utilize class="dropdown-trigger" e data-target para criar dropdowns. Inicialize com JavaScript.

Collapsibles

Crie seções expansíveis com div class="collapsible" e li class="collapsible-header" para o cabeçalho.

Carrosséis

Utilize div class="carousel" para criar carrosséis e inicialize com JavaScript

Parallax

Adicione o efeito de paralaxe com div class="parallax" e inclua uma imagem dentro da div.
